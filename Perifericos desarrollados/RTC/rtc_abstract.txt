The RTC project is a simple program for the NXP 'LPC1768' 
microcontroller.
The current time value is stored in variable of current_time, 
and can be displayed in Watch& Call Stack Window.

Example functionality:                                                   
 - Clock Settings:
   - XTAL  =  12 MHz
   - PLL0  = 288 MHz
   - CCLK  =  72 MHz
